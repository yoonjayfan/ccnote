# CCNOTE

基于 Claude Code 的 Markdown 笔记管理工具。通过斜杠命令 `/cn` 快速保存和查找笔记，支持智能融合写入，并提供浏览器端的笔记阅读器。

## 功能特点

- 输入 `/cn <内容>` 即可保存或查找笔记
- 智能融合：理解文档结构，将内容插入语义最合适的位置
- 浏览器阅读器：左侧目录 + 右侧内容，支持代码高亮
- 自动维护笔记目录索引

## 快速开始

1. 用 VSCode 打开本项目
2. 安装推荐扩展（Claude Code、Live Server）
3. 在 Claude Code 中输入 `/cn 你想记录的内容`

## 使用说明

详细使用方法请查看 [CCNOTE说明书](notes/CCNOTE说明书.md)。

## 浏览笔记

右键点击 `home.html`，选择 "Open with Live Server" 即可在浏览器中阅读所有笔记。
