# CCNOTE 使用说明

> 2026-05-19

## 什么是 CCNOTE

CCNOTE 是一个基于 Claude Code 的 Markdown 笔记管理工具。通过斜杠命令 `/cn` 快速保存和查找笔记，支持智能融合写入，并提供浏览器端的笔记阅读器。

## 前置要求(必须有)

- VSCode
- VSCode 的 Claude Code 扩展
- Git 仓库(否则不会提示 /cn 命令)
- VSCode + Live Server 扩展（用于浏览器预览）

## 浏览器中查看笔记的方法

1. 在 VSCode 中右键点击 `home.html`
2. 右键选择 "Open with Live Server"
3. 一般会在浏览器中自动弹出地址`http://127.0.0.1:5500/home.html`
4. 侧边栏宽度可拖拽调整

## 核心命令

### 保存笔记

```
/cn 你想记录的任何内容,也可以描述你要做的操作
```

系统会询问保存位置：
- **A** 写入上次编辑的文档（快速追加）
- **B** 创建新笔记
- **C** 查找相关笔记后写入

### 查找笔记

```
/cn 什么是 XXX
/cn 查找关于 Python 的笔记
```

包含疑问词或查询动词时自动进入查询模式，只读不修改。

## 写入方式

### 融合

将内容插入到已有文档中语义最合适的位置，保持原文结构不变，末尾标注日期 `[YYYY-MM-DD]`。

### 追加

在文档末尾添加分隔线和新内容：

```markdown
---
> 2026-05-19

新增的内容写在这里
```

## 文件结构

| 文件 | 用途 |
|------|------|
| `notes/` | 所有笔记存放目录 |
| `index.md` | 笔记目录索引，保存时自动更新 |
| `home.html` | 浏览器笔记阅读器 |
| `.claude/commands/cn.md` | `/cn` 命令的 prompt 模板 |
| `CLAUDE.md` | 项目级指令和格式规则 |

## 小贴士

- 文件名建议用中文，方便检索
- 每次保存后目录会自动更新，无需手动维护 `index.md`
- 融合模式适合往已有主题补充内容，追加模式适合记录独立条目
